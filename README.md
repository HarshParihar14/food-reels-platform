# 🍽️ Food Reels Platform

A full-stack MERN application where users can explore short food videos (reels), like/save content, and food partners can upload and manage videos.

---

## 🚀 Features

- 🎥 Short video reels feed (like Instagram)
- ❤️ Like / 🔖 Save functionality
- 👤 User Authentication (Login/Register)
- 🍴 Food Partner Dashboard
- 📤 Upload food videos
- 📱 Responsive UI (Mobile-first design)
- 🔄 Real-time UI updates

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS (Custom UI)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other
- Axios
- JWT Authentication
- Multer (file upload)
